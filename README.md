Dit is een To-Do Dashboard waar je een overzicht hebt van je taken en makkelijk alles kan onthouden.
Het gebruiken van de website is simpel:
1. Download de bestanden
2. Ga naar http://localhost:3000/login.php
3. Log in of maak een account aan
4. Voeg taken toe en houdt een overzicht bij.
